# React-приложение для управления персоналом

Полнофункциональный SPA для учета сотрудников компании с возможностью полного цикла работы с данными:

## 🔧 Основной функционал:
- CRUD-операции – добавление новых сотрудников, редактирование данных, удаление из системы
- Динамическая статистика – отображение общего числа сотрудников, количества премированных
- Система фильтрации – "все сотрудники", "на повышение", "З/П > 1000$"
- Поиск по сотрудникам – быстрый поиск по имени
- Управление статусами – назначение премии (нажмите на 🍪), отметка о повышении (нажмите на имя сотрудника), изменение зарплаты (нажмите на поле зарплаты)

## ⚙️ Технические особенности:
- Адаптивный интерфейс для всех устройств
- Управление состоянием приложения
- Работа с формами и валидацией
